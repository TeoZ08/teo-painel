# Referência de dashboard

Origem: assets fornecidos pelo usuário nesta conversa. Não são declarados como autoria deste repositório.

Uso: referência de composição, sidebar, densidade, tabela de projetos, controles e formas vetoriais. A implementação adapta esses elementos para o `teo-painel` em React e CSS; não usa capturas como interface funcional, nem replica nome, pessoas, métricas corporativas ou conteúdo fictício.

Elementos adaptados: navegação lateral expansível, item ativo em pílula, botão de ação principal, cabeçalho de busca, indicadores úteis e tabela responsiva.
